vue-awesome-countdown

适用于Vue 2.5.0+的高性能高精度倒计时插件。

快速上手 →

易用

代码简洁的同时有着丰富的插槽(slots)和方法(methods)，可应对多种场景。

高精度

整个流程下来与预期结束时间误差仅为1~3毫秒。

轻量级

gzip压缩后不到2kb。

<h1>倒计时：
  <countdown :left-time="101000"> <!-- 101秒 -->
    <span
      slot="process"
      slot-scope="{ timeObj }">
        {{ timeObj.ceil.s }}
      </span>
      <span slot="finish">干完!</span>
  </countdown>
</h1>

注意

请确保你的 Node.js 版本 >= 8，
Vue.js 版本 >= 2.5.0。
